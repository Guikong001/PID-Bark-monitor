# PID-Bark-monitor
使用Bark信息推送软件监测Linux系统进程运行状态。

用法: sljiankong [选项]...
选项:
  -p PID列表    添加pid编号，可以添加多个，使用;隔开，如添加多个PID，请把所有PID使用英文单引号包裹
  -t 标题        更换推送标题
  -c 监控间隔    监控间隔（单位：秒）
  -w 进程间隔    当前正在运行进程的监控间隔（单位：秒）
  -m 密钥        设置Bark推送服务的密钥
  -s 主机        设置为你的自定义Bark服务器
  -h, --help     显示此帮助信息并退出
  -v, --version  显示版本信息并退出
